# 写在前面

> ✍️Writing code is like liberty art.

阅读团队的代码规范，并且遵守它，的确能够统一团队代码风格。

但是，要想真正地编写出可维护、可拓展、高灵活性的，也就是我们口中的「好代码」。仅靠此代码规范，还远远不够。

你需要学习体验设计模式、重构思想、各种软件工程的概念与原则。亲身地去体会，感受编写「优雅的」代码给我们带来的好处。
